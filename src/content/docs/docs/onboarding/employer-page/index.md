---

Every company or brand publishing jobs on FashionUnited has their own employer page. An Employer Page is a basic page, also known as "Work at" or "dedicated" page. It contains a preset banner image, the company/brand logo, company description and job overview.

## Setup

We will set up the page using the logo from your first job posting and will add a company description (taken from either the job description or your own website).

## Updating your employer page

Logo and description can be changed by contacting [jobs@fashionunited.com](mailto:jobs@fashionunited.com). Send the link to your employer page along with the information to update:

- **Logo** — Your logo as SVG (or PNG) file
- **Company description** — A short description of your company (who are you?)
- When posting jobs internationally, the company description can be displayed in different languages (per country a job is published on)

For Premium Employer branding page options, see [Premium Employer](/docs/onboarding/premium-employer/).
