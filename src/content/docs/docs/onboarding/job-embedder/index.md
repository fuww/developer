There is the possibility to embed jobs that are posted on our job board on your website. This is ideal for clients who do not have their own career page or do not have an ATS system.

Jobs that are manually posted on FashionUnited automatically go live on your website as well. Via the Dashboard you are able to manage the jobs (publish, unpublish, repost and edit). FashionUnited serves as "the source" — the connection between our job board and your website (we are your ATS).

With this API you can easily display jobs on your own website while the jobs are hosted by us. This means you do not need to set up an infrastructure to handle vacancies, and it also generates far more traffic than publishing jobs on your website alone.

## Setup

The setup of the embedder is something you need to do yourself. We are not able to support you with this on a technical level, since all the work needs to happen on your end.

You need to embed a code snippet on your website. The Customer Service team can create this code for you ([jobs@fashionunited.com](mailto:jobs@fashionunited.com)). To place this on your website, all you need to do is paste the code on any page you want. It is easier than embedding a YouTube video.

### Bare minimum code

Add locale and profile ID (to be generated by FashionUnited). Styling can be adapted to match any site design:

```html
<div
  class="fu-embed-jobs"
  data-locales=""
  data-profile_id=""
  data-component="CompanyJobs">
  <script src="https://fashionunited.com/global-assets/jobs-embed/embed.js" async=""></script>
</div>
```

### With optional fields

```html
<div
  class="fu-embed-jobs"
  data-locales="en-US,nl-NL"
  data-category="Design & Creative"
  data-profile_id="2HycEba6EMxMYbyJN"
  data-keywords="fashionweek"
  data-limit="20"
  data-skip="15"
  data-component="CompanyJobs">
  <script src="https://fashionunited.com/global-assets/jobs-embed/embed.js" async=""></script>
</div>
```

## Attributes

| Attribute | Description |
|---|---|
| `data-locales` | Comma-separated list of locales to display. All locales if omitted. |
| `data-category` | Category filter. Values: "Design & Creative", "Internships", "Other", "Product & Supply Chain", "Retail Management & In-store", "Sales & Marketing" |
| `data-profile_id` | MongoDB `_id` of the profile to display (generated by FashionUnited) |
| `data-keywords` | Keywords to filter on (same as the search field in the job board) |
| `data-limit` | Number of jobs to display (default: 10) |
| `data-skip` | Number of jobs to skip (for pagination) |
| `data-component` | Display component: `CompanyJobs` (default — company page), `HomePageJobs` (home pages), `SearchJobs` (adds search box) |

## Examples

### Third-party site

```html
<div
  class="fu-embed-jobs"
  data-component="CompanyJobs"
  data-locales="en-US,nl-NL"
  data-limit="5"
  data-profile_id="Y6Maua8gHEFRHuQTN">
</div>
```
